DobierzFalownik.pl - Front-End

Aplikacja internetowo umożliwiająca dobór falowników z oferty sklepu internetowego klienta na podstawie określonych parametrów takich jak: napięcie zasilania, napięcie wyjściowe, moc falownika, prąd wyjściowy lub stopień ochrony obudowy IP.

Prezentowana aplikacja składa się z części użytkownika (pozwalającej na filtrowanie falowników, wyświetlanie informacji na ich temat oraz przejście po kliknięciu odnośnika do sklepu klienta) oraz części administratora umożliwiającej dodawanie falowników, ich edycję oraz usówanie modeli wycofanych z oferty.

Dostęp do części administracyjnej możliwy jest wyłącznie po zalogowaniu. W celu autoryzacji użytkownika program wykorzystuje mechanizm "Json Web Token - JWT".

  • Technologia : Angular 9, Material Design, API, JWT, Routing
  • GitHub : https://github.com/DamianLew23/DobierzFalownik-Angular
  • Live : https://dobierzfalownik.pl
